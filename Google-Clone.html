Google Clone

Google Clone to Showcase MERN Stack

This is a Google Clone that I built using the MERN tech stack. I used firebase to facilitate the backend functionality. Furthermore I used react Hooks for the first time which I really enjoy. I prefer hooks to props and high-order components. I am thankful that I did not have to deal with this long when I was learning react and only had to deal with a few instances of "wrapper hell" of layers of providers, consumers, and HOC among others that really clutter the code and affect readability. I think they have done an excellent job trying to solve this with react hooks.
